# Color Audit Screenshot Saver v1.5.1

## 主な機能

- サイトのカラーコード埋め込み資料を作成する作業を自動化します。
- 拡張を導入し、資料を作成したいサイトで実行するだけで、png形式の資料が生成され、そのままダウンロードできます。
- カラーコード資料作成作業とは、サイトのスクリーンショット画像に対して、どの要素にどのカラーコードが使用されているかを手作業で書き込む作業です。
- この資料を用いることで、上位の立場の人がサイトを確認する際に、どの要素にどのカラーコードが使われているのかを視覚的に把握できます。
- 網羅性を意識して作成していますが、すべての要素を完全に取得できるとは限らないため、大変申し訳ありませんが、その場合は手動で補完してください。
