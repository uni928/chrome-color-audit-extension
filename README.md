# Color Audit Screenshot Saver v1.5.1

## 主な機能

- サイトのカラーコード埋め込み資料を作成する作業を自動化します。
- 拡張を導入して、資料を作りたいサイトで実行するだけで png の資料が完成し、ダウンロード画面が出現します。
- カラーコード資料とは、サイトのスクリーンショット画像に対して、どの要素にどのカラーコードを使用しているかの情報を手書きで入力する作業です。
- カラーコード資料を見れば、上の立場の人が作成したサイトをチェックする際に、どの要素がどのカラーコードを使っているのか視覚的にわかります。
- 網羅性を意識して作成していますが、全ての要素に対応できるとは限りません。手動で追記する作業が必要になる可能性がある事は申し訳ありません。

## ライセンスについて

- ライセンスは MIT です。
- MIT ライセンスの性質上、この拡張で生成した成果物(png資料)に、そのライセンスは確実に引き継がれないです。また、そうであると明言しておきます。

## 主な追加
- 同じ番号が5箇所以上になる場合は、離れている代表5箇所だけに表示
- それ以外の同番号箇所には数字を描かない
- 1つのCSSラベルにつき、番号は1つで固定
- 新モード追加:
  - 番号 + 下部凡例
  - 詳しくその場所に表示

## 補足
- 今回の「詳しくその場所に表示」は、同じCSSについて代表箇所だけに詳細ラベルを出します
